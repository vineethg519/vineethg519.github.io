Collaboration Tool

Windows Desktop Stand-alone Application using C# language.

A complete package consisting a chat tool, task manager, shared calendar and cloud service using google drive. Users can sign up and create teams so that all team members in a group can collaboratively work together. Members has ability of one to one chat or group chat with entire team. I have used Socket class from .net library for communications in chat module. Utilized AWS EC2 instance for hosting the serve online. This server handles all the client requests and responses for chatting. Each user can connect to this server by a static IP which is given by Amazon. All the data like chat history, messages, multimedia data is stored in Microsoft’s sql server. I connected our application to this using a url connection string provided by Amazon which acts as an endpoint. Amazon’s S3 service is used for storing the data online securely. Implemented SHA512 hashing algorithm for securely storing the passwords of the users.Using Google’s oauth 2.0 allowing users to access their drive account securely. Users can upload, download and delete their google drive files within the phoenix tool.

  • Date: June 2017 – Dec 2017
  • Role: Windows Desktop Application Developer
